##### 1.小鱼的航程（改进版）

> 有一只小鱼，它平日每天游泳 250 公里，周末休息（实行双休日)，假设从周 x(1≤x≤7) 开始算起，过了 n(n≤106) 天以后，小鱼一共累计游泳了多少公里呢？
>
> 输入格式
>
> 输入两个整数x,n(表示从周x算起，经过n天）。

**星期问题 —— 类似过几天后是星期几**

**思路：过n天— 循环，每天都判断是不是周末，是周末航程不增加。**

```C++
#include<iostream>
using namespace std;
int main(){
    int x,n,m=0;
    cin>>x>>n;
    for(int i=1;i<=n;i++){	//注意这里的分支是怎么写的
        if(x!=6 && x!=7){		//这个单独一支，用于做判断和航程计算
            m+=250;
        }
        if(x==7){			//这里的if-else 用于星期数（x）的变更
            x=1;
        }
        else x++;
    }
    cout<<m<<endl;
    return 0;
}
```

```C++
#include<iostream>
#include<math.h>
using namespace std;
int main(){
    int tr[3],temp;
    for(int i=0;i<3;i++){
        cin>>tr[i];
    }
    for(int i=0;i<3-1;i++){
        for(int j=0;j<3-i-1;j++){
            if(tr[j]>tr[j+1]){
                temp=tr[j];
                tr[j]=tr[j+1];
                tr[j+1]=temp;
            }
        }
    }
    if(tr[0]+tr[1]<=tr[2]) {
        cout<<"Not triangle";
        return 0;}	// 不是三角形就不用接着继续判断了，直接结束
    if(pow(tr[0],2)+pow(tr[1],2)==pow(tr[2],2)) cout<<"Right triangle"<<endl;
    else if(pow(tr[0],2)+pow(tr[1],2)>pow(tr[2],2) && tr[0]+tr[1]>=tr[2]) cout<<"Acute triangle"<<endl;
    else cout<<"Obtuse triangle"<<endl;
    if(tr[0]==tr[1] || tr[1]==tr[2]) cout<<"Isosceles triangle"<<endl;
    if(tr[0]==tr[1] && tr[0]==tr[2]) 
        cout<<"Equilateral triangle";
    return 0;
}
```

##### 4.ABC——字符串数组

> 三个整数分别为 A,B,C。这三个数字不会按照这样的顺序给你，但它们始终满足条件：A<B<C。为了看起来更加简洁明了，我们希望你可以按照给定的顺序重新排列它们。
>
> **【输入格式】**第一行包含三个正整数 A,B,C，不一定是按这个顺序。这三个数字都小于或等于 100。第二行包含三个大写字母 A、B和 C（它们之间**没有**空格）表示所需的顺序。
>
> **【输出格式】**在一行中输出 A，B和 C，用一个 ` `（空格）隔开。

```C++
#include<iostream>
#include<algorithm>
#include<string.h>
using namespace std;
int main(){
    int num[3],a,b,c;
    char str1[4];
    for(int i=0;i<3;i++){
        cin>>num[i];
    }
    sort(num,num+3);
    a=num[0];
    b=num[1];
    c=num[2];
    cin>>str1;
    if(strcmp(str1,"ABC")==0) cout<<a<<" "<<b<<" "<<c;
    if(strcmp(str1,"ACB")==0) cout<<a<<" "<<c<<" "<<b;
    if(strcmp(str1,"BAC")==0) cout<<b<<" "<<a<<" "<<c;
    if(strcmp(str1,"BCA")==0) cout<<b<<" "<<c<<" "<<a;
    if(strcmp(str1,"CAB")==0) cout<<c<<" "<<a<<" "<<b;
    if(strcmp(str1,"CBA")==0) cout<<c<<" "<<b<<" "<<a;
    return 0;
}
```

##### 5. ISBN号码

> ISBN码包括9位数字、11位识别码和3位分隔符，其规定格式如`x-xxx-xxxxx-x`，其中符号`-`就是分隔符（键盘上的减号），最后一位是识别码。识别码的计算方法如下：首位数字乘以1加上次位数字乘以2……以此类推，用所得的结果  mod11，所得的余数即为识别码，如果余数为10，则识别码为大写字母X。
>
> 例如ISBN号码`0-670-82162-4`中的识别码4是这样得到的：对`067082162`这9个数字，从左至右，分别乘以1,2,...,9再求和，即0×1+6×2+……+2×9=158，然后取158mod11的结果4作为识别码。
>
> 你的任务是编写程序判断输入的ISBN号码中识别码是否正确，如果正确，则仅输出`Right`；如果错误，则输出你认为是正确的ISBN号码。

```C++
#include<iostream>
#include<string>
using namespace std;
int main(){
    char num[14];
    int a;
    cin>>num;
    a=((num[0]-'0')*1+
    (num[2]-'0')*2+(num[3]-'0')*3+(num[4]-'0')*4+
    (num[6]-'0')*5+(num[7]-'0')*6+(num[8]-'0')*7+(num[9]-'0')*8+(num[10]-'0')*9)%11;
    if(a==(num[12]-'0')) cout<<"Right"<<endl;
    else {
        for(int i=0;i<12;i++){
            cout<<num[i];
        }
        cout<<a;
    }
    return 0;
}
// 第一次代码50分
// 忽略了题目当识别码为10时，要表达为X，这一要求
```
