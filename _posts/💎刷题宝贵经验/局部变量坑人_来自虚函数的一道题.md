# 基类指针数组



**题目来源：PTA 图书音像出租管理**

一个图书音像店出租图书和磁带。
给出下面一个基类的框架：

```
class Publication
{
protected:
string title;//名称
float price;//原价
int day;//租期
public:
virtual void display()=0;//打印价格清单
}
```

以Publication为基类，构建Book和Tape类。
生成上述类并编写主函数，要求主函数中有一个基类Publication指针数组，数组元素不超过10个。
`Publication *pp[10];`
主函数根据输入的信息，相应建立Book, Tape类对象。
它们的原始租金为：租期\*1.2。
另外，实际收取的租金不超过2倍的租品估价。
Book的估价为：新旧程度*原价。
Tape的估价为：原价/（1+已出租次数/3）。
**输入格式：**每个测试用例占一行，第一项为租品类型，1为Book,2为Tape.接下来为名称、原价、租期。最后一项Book是新旧程度（0.1至1），Tape是已出租次数。以0表示输入的结束。
**要求输出：**名称，原始租金(小数点后保留1位小数)，如果原始租金大于2倍租品估价，则同时给出实际应收取的租金(小数点后保留1位小数)，并在最后标明R。 



输入样例

```
1 AAA 19.5 3 0.5
1 BBB 9.5 2 0.1
2 AA 10 2 0
2 DDDD 12.5 2 38
1 FFF 42 3 0.1
0
```

输出样例

```
AAA 3.6
BBB 2.4 1.9 R
AA 2.4
DDDD 2.4 1.8 R
FFF 3.6
```



## 先解决WA问题

```C++
#include<iostream>
#include<iomanip>
using namespace std;

class Publication{
    protected:
        string title;   // 名称
        float price;    // 原价
        int day;        // 租期
    public:
        Publication(string n,float p,int d){
            this->title = n;
            this->price = p;
            this->day = d;
        }
        virtual void display() = 0;
};
class Book:public Publication{
    float level;    // 新旧程度
    public:
        Book(string n,float p, int d, float l):Publication(n,p,d){
            this->level = l;
        }
        float getValuation(){       // 估价
            return level*price;
        }
        float getOriginalPrice(){      // 原始租金
            return day*1.2;
        }
        void display(){
            cout << fixed << setprecision(1);
            if(getOriginalPrice() > getValuation()*2){
                cout << title << " " << getOriginalPrice() 
                  << " " << getValuation() << " R"<<endl;
            }
            else
                cout << title << " " << getOriginalPrice()<<endl;
        }
};

class Tape:public Publication{
    int time;       // 出租次数 
    public:
        Tape(string n,float p, int d, int t):Publication(n,p,d){
            this->time = t;
        }
        float getValuation(){       // 估价
            return price/(1+time/3);
        }
        float getOriginalPrice(){      // 原始租金
            return day*1.2;
        }
        void display(){
            cout << fixed << setprecision(1);
            if(getOriginalPrice() > getValuation()*2){
                cout << title << " " << getOriginalPrice() << " " 
                  << getValuation() << " R"<<endl;
            }
            else
                cout << title << " " << getOriginalPrice()<<endl;
        }
};

int main(){
    int choose;
    Publication* pp[10] = {0};
    int i = 0;
    string name;
    float price;
    int day;
    float level;
    int time;
    while(true){
        cin>>choose;
        if(choose == 0)
            break;
        if(choose == 1){
            cin>>name>>price>>day>>level;
            Book book(name,price,day,level);		// 问题出在这里
            pp[i] = &book;
            i++;
        }
        if(choose == 2){
            cin>>name>>price>>day>>time;
            Tape tape(name,price,day,time);
            pp[i] = &tape;
            i++;
        }
    }
    for(int j = 0; j<i; j++){
        pp[j]->display();
    }
    return 0;
}
```

报错信息：`libc++abi: Pure virtual function called!`

**大概的问题是在类构造函数或析构函数中调用了虚函数引起的。**



### 问题原因：

            Book book(name,price,day,level);		// 问题出在这里
            pp[i] = &book;
            i++;

这里我在if里面创建的变量，是局部变量，作用域只在if语句内，存储在栈中，在if语句外内存被释放，这样pp[i]指向的内存就为空了，这样就有问题了。

    for(int j = 0; j<i; j++){
        pp[j]->display();
    }





### 改进1⃣️

> 使用new来创建变量，这样变量就存储在堆中了（堆是最后释放的）

```C++
int main(){
    int choose;
    int i = 0;
    string name;
    float price;
    int day;
    float level;
    int time;
    while(true){
        cin>>choose;
        Publication *pp;
        if(choose == 0)
            break;
        if(choose == 1){
            cin>>name>>price>>day>>level;
            pp = new Book(name,price,day,level);
        }
        if(choose == 2){
            cin>>name>>price>>day>>time;
            pp = new Tape(name,price,day,time);
        }
        pp->display();
        delete pp;
    }
    return 0;
}
```

报错信息：`zsh: trace trap`	PS：这个报错信息只有我这里有，可能是电脑配置问题。这样写是没有问题的。

**实现思路：** 使用基类指针，new一个派生类对象赋值给基类指针，然后`pp->display();`  调用虚函数（动态联编）执行对象的函数，输出完记得要删除掉指针，（因为每一次一进while，又会创建一个pp指针）



## 学习好的处理方法

> AC code

```C++
#include<iostream>
#include<iomanip>
using namespace std;

class Publication{
    protected:
        string title;   // 名称
        float price;    // 原价
        int day;        // 租期
    public:
        Publication(string n,float p,int d){
            this->title = n;
            this->price = p;
            this->day = d;
        }
        virtual void display() = 0;
};
class Book:public Publication{
    float level;    // 新旧程度
    public:
        Book(string n,float p, int d, float l):Publication(n,p,d){
            this->level = l;
        }
        void display(){
            cout << fixed << setprecision(1);
            if(day*1.2 > level*price*2){
                cout << title << " " << day*1.2 << " " << level*price*2 << " R"<<endl;
            }
            else
                cout << title << " " << day*1.2 << endl;
        }
};

class Tape:public Publication{
    int time;       // 出租次数 
    public:
        Tape(string n,float p, int d, int t):Publication(n,p,d){
            this->time = t;
        }
        void display(){
            cout << fixed << setprecision(1);
            if(day*1.2 > (price/(1+(float)time/3))*2){
                cout << title << " " << day*1.2 << " " 
                  << (price/(1+(float)time/3))*2 << " R"<<endl;
            }
            else
                cout << title << " " << day*1.2 <<endl;
        }
};

int main(){
    int choose;
    cin >> choose;
    string name;
    float price;
    int day;
    float level;
    int time;
    for(int i =0; choose != 0; i++){
        Publication *pp[10];
        if(choose == 1){
            cin>>name>>price>>day>>level;
            pp[i] = new Book(name,price,day,level);
        }
        if(choose == 2){
            cin>>name>>price>>day>>time;
            pp[i] = new Tape(name,price,day,time);
        }
        pp[i]->display();
        cin>>choose;
    }
    return 0;
}
```

好的处理方法：

- 之后需要使用的变量，不要在if中创建！一出了if语句就不留情的给你删啰！
- 如果要在if中实现选择的创建变量，那就用new，把它整在堆里。

这个题目就非常好的体现了虚函数实现动态联编的优点，🐮  搞出虚函数这玩意真是太有才了！
