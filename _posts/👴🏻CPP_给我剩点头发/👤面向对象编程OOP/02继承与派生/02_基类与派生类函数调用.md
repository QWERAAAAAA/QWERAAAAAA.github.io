## 1. 构造函数和析构函数的调用



> 前面我们说基类的成员函数可以被继承，可以通过派生类的对象访问，但这仅仅指的是普通的成员函数，**类的构造函数和析构不能被继承**。构造函数不能被继承是有道理的，因为即使继承了，它的名字和派生类的名字也不一样，不能成为派生类的构造函数，当然更不能成为普通的成员函数。



在设计派生类时，**对继承过来的成员变量的初始化工作也要由派生类的构造函数完成**，但是大部分基类都有 private 属性的成员变量，它们在派生类中无法访问，更不能使用派生类的构造函数来初始化。



这种矛盾在C++继承中是普遍存在的，解决这个问题的思路是：==**在派生类的构造函数中调用基类的构造函数。**==



🔎 因为基类构造函数不会被继承，不能当做普通的成员函数来调用。换句话说，**只能将基类构造函数的调用放在函数头部，不能放在函数体中。**

```
// 错误示范
Derived(int x, int y, int z){	
            Base1(x)；
            Base2(y)；
            d = z;
        }
```



> 在定义派生类构造函数时，需要调用基类的构造函数，初始化基类成员。

```c++
// 多重继承

#include<iostream>
using namespace std;

class Base1{        // 基类1
    protected: int data1;
    public:
        Base1(int a = 0){
            data1 = a;
            cout<<"Base1 Constructor\n";
        }
        ~Base1(){ cout << "Base1 Destructor\n";}
};

class Base2{        // 基类2
    protected: int data2;
    public:
        Base2(int a = 0){
            data2 = a;
            cout<<"Base2 Constructor\n";
        }
        ~Base2(){ cout << "Base2 Destructor\n";}
};

class Derived: public Base1, public Base2 {
    int d;
    public:
  
        //看这里👀，派生类的构造函数的定义格式
        Derived(int x, int y, int z): Base1(x), Base2(y){		// 在定义派生类的构造函数时调用了基类的构造函数
            d = z;
            cout << "Derived Constructor\n";
        }
        ~Derived(){ cout << "Derived Destructor\n";}
        void Show(){
            cout << data1 << ',' << data2 << ',' << d << endl;
        }
};

int main(){
    Derived c(1,2,3);
    c.Show();
    return 0;
}
```

运行结果：

```output
Base1 Constructor
Base2 Constructor
Derived Constructor
1,2,3
Derived Destructor
Base2 Destructor
Base1 Destructor
```



- **派生类构造函数的执行顺序：**

​		首先依次调用基类构造函数，然后执行<派生类自身的构造函数>

- **析构函数的执行顺序：**

​		和构造函数相反



## 2. 对象成员构造函数和析构函数的调用顺序

> 情况：派生类中包括基类成员和对象成员。
>
> 怕你概念忘了）对象成员是什么：类的实例是对象，另一个类中用对象作为数据成员，那这个数据成员就是对象成员。
>
> 所以，类内使用对象成员又可称为“**组合**”方式

```c++
#include<iostream>
using namespace std;

class Base1{        // 基类1
    protected: int data1;
    public:
        Base1(int a = 8){
            data1 = a;
            cout<< data1 <<", Base1 Constructor\n";
        }
        ~Base1(){ cout << data1 << ", Base1 Destructor\n";}
};

class Base2{        // 基类2
    protected: int data2;
    public:
        Base2(int a = 9){
            data2 = a;
            cout << data2 <<", Base2 Constructor\n";
        }
        ~Base2(){ cout << data2 << ", Base2 Destructor\n";}
};

class Derived: public Base1, public Base2 {
    int d;
    Base1 b1,b2;		// 包含两个Base1类的对象成员
    public:
        //看这里👀，含对象成员的派生类的构造函数的定义格式
        Derived(int x, int y, int z): Base1(x), Base2(y), b1(x+y), b2(x+z) {
            d = z;
            cout << "Derived Constructor\n";
        }
        ~Derived(){ cout << "Derived Destructor\n";}
        void Show(){
            cout << data1 << ',' << data2 << ',' << d << endl;
        }
};

int main(){
    Derived c(1,2,3);
    c.Show();
    return 0;
}
```

运行结果：

```
1, Base1 Constructor	//调用基类的构造函数
2, Base2 Constructor
3, Base1 Constructor	//调用成员对象的构造函数
4, Base1 Constructor
Derived Constructor		//调用派生类的构造函数
1,2,3
Derived Destructor
4, Base1 Destructor
3, Base1 Destructor
2, Base2 Destructor
1, Base1 Destructor
```
