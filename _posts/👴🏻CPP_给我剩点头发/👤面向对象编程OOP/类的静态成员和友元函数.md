### 类的静态成员与友元函数 —— 实例：银行账户



​    类的静态成员和友元函数

​    静态成员：共享

​    友元函数：特权 



```
class Account{

    static int count;						// 账户数量
    static float InterestRate;  // 利润
    string  _accno, _accname;		// 账户与户名
    float _balance;             // 账户余额

    public:
        Account(string accno, string name, float balance);
        ~Account();
        void Deposit(float amount);     // 存款
        void Withdraw(float amount);    // 取款
        float GetBalance();             // 获取账户余额
        void Show();                    // 显示账户基本信息
        static int GetCount();          // 获取账户数量
        static float GetInterestRate();    // 获取利润信息      
};
```



#### 要求如下：

1) 实现该银行账户类

2) 为账户类Account增加一个友元函数，实现账户结息，要求输出结息后的余额（结息余额=账户余额+账户余额*利率）。

​			`友元函数声明形式为 friend void Update(Account& a);`

3) 在main函数中，定义一个Account类型的指针数组，让每个指针指向动态分配的Account对象，并调用成员函数测试存款、取款、显示等函数，再调用友元函数测试进行结息。创建Account指针数组和对象的参考代码如下：

​			`Account **pAccounts = new Account*[n]` 

​			`//创建一个Account类型的指针数组，数组中的每个元素为Account*类型的指针，其中n为需要创建的账户数`

​			`For(int i=0; i<n; i++) pAccounts[i] = new Account(no, name, bal);` 

​			`//为每个Account指针动态创建一个Account对象，并让该指针指向该对象`
