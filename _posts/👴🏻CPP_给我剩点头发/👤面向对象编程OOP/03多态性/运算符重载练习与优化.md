## 题目描述

```
编程练习：运算符重载

定义一个分数类，其数据成员为分子和分母，用成员函数重载运算符=、+和-，以实现分数的赋值、相加和相减。用友元函数重载运算符*和/，以实现分数的相乘和相除。要求进行各种运算后，得到的分数是最简的。请编写一个完整的程序，测试各种运算符重载的正确性。假定分子和分母都是整数。

提示：为了得到最简分数可在构造函数中对分子分母进行约简，进而需要另外编写一个求两个数的最大公约数的成员函数。
```

```C++
#include<iostream>
#include<math.h>
using namespace std;

// 还未实现错误输入和分母为0情况的报错

class Fraction{
    int child, mom;
    public:
        Fraction(int c = 1, int m = 1){     // 构造函数
            child = c;
            mom = m;
        }

        // 因为不能保证运算后的结果也是最简的，还要定义一个约分函数
        void reduction(){
            int a, b;
            if(child == 0){
                mom = 1;
                return ;
            }
            if(abs(child) > abs(mom))
                b = mom;
            else    b = child;
            b = abs(b);
            for(a = b;a>1;a--){
                if(mom % a == 0 && child % a == 0){
                    child /= a;
                    mom /= a;
                    return ;
                }
            }
        }
        // 在运算过程，需要用到通分
        void commond(Fraction &f){
            child = child * f.mom;
            mom = mom * f.mom;
        }

        void display(){
            reduction();
            if(child == 0)
                cout << '0';
            else if(mom == 1)
                cout << child;
            else   
                cout << child << '/' << mom;
        }

        Fraction operator=(const Fraction  &f){
            child = f.child;
            mom = f.mom;
            return *this;
        }
        Fraction operator+(Fraction f){
          // 这里不能用const Fraction &f，因为用了const就不能修改f了
            Fraction result(child,mom);
            Fraction tempf(f.child,f.mom);     
            f.commond(result);
            result.commond(tempf);
            result.child += f.child;
            result.reduction();
            return result;
        }
        Fraction operator-(Fraction f){
            Fraction result(child,mom);
            Fraction tempf(f.child,f.mom);
            f.commond(result);
            result.commond(tempf);
            result.child -= f.child;
            result.reduction();
            return result;
        }


        friend Fraction operator*(const Fraction &f1, const Fraction &f2);
        friend Fraction operator/(const Fraction &f1, const Fraction &f2);
};

Fraction operator*(const Fraction &f1, const Fraction &f2){
    int child = f1.child * f2.child;
    int mom = f1.mom * f2.mom;
    return Fraction(child, mom);       
}

Fraction operator/(const Fraction &f1, const Fraction &f2){
    Fraction f3(f2.mom,f2.child); 
    int child = f1.child * f3.child;
    int mom = f1.mom * f3.mom;
    return Fraction(child,mom);
}
```

## 代码优化

```C++
#include<iostream>
#include<math.h>
using namespace std;

class Fraction{
    int child, mom;
    void commond(Fraction &f){      // 通分函数只是一个过程函数，设为私有，不向外界提供通分函数的接口。
            child = child * f.mom;
            mom = mom * f.mom;
        }
    public:
        Fraction(int c = 1, int m = 1){     // 构造函数
            child = c;
            mom = m;
            if(m == 0){
                cout << "分母不能为0" << endl;
                exit(1);				// exit(1)表示错误终止，exit(0)表示正常终止。
            }
            int a, b;
            if(child == 0){
                mom = 1;
                return ;
            }
            if(abs(child) > abs(mom))
                b = mom;
            else    b = child;
            b = abs(b);
            for(a = b;a>1;a--){
                if(mom % a == 0 && child % a == 0){
                    child /= a;
                    mom /= a;
                    return ;
                }
            }
        }
        
        void display(){
            if(child == 0)
                cout << '0';
            else if(mom == 1)
                cout << child;
            else   
                cout << child << '/' << mom;
        }

        Fraction operator=(const Fraction  &f){
            child = f.child;
            mom = f.mom;
            return *this;
        }
        Fraction operator+(const Fraction &f){         
            return Fraction(child*f.mom + mom*f.child, mom*f.mom);
        }
        Fraction operator-(const Fraction &f){
            return Fraction(child*f.mom - mom*f.child, mom*f.mom);
        }


        friend Fraction operator*(const Fraction &f1, const Fraction &f2);
        friend Fraction operator/(const Fraction &f1, const Fraction &f2);
};

Fraction operator*(const Fraction &f1, const Fraction &f2){
    return Fraction(f1.child * f2.child, f1.mom * f2.mom);       
}

Fraction operator/(const Fraction &f1, const Fraction &f2){
    return Fraction(f1.child*f2.mom,f1.mom*f2.child);
}
```



### 优化1: 设为私有or公有

> 一定要有产品思维！

- **将通分函数设为私有成员函数**

​		因为，通分函数只是一个过程函数，设为私有，不向外界提供通分函数的接口。（接口并不是越多越好，就像洗衣机上的按钮不是越多越好！）



### 优化2: 在构造函数中实现化简

只要每次返回`return Fraction(child, mom)`即可。这样每次返回都会调用Fraction的构造函数，实现化简。



### 优化3: 简洁的运算符重载函数

直接return完成操作，多么的简洁、干净、利落。
