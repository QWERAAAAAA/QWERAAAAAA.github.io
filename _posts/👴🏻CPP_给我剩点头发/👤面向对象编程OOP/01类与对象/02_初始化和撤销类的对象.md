## 初始化和撤销对象

> 主要内容：构造函数和析构函数、拷贝构造函数
>
> 特殊的成员函数。
>
> ==涉及的一个重点内容：内存空间被释放两次==

▪️结构体与类的初始化区别

结构体变量成员的初始化可以在定义结构体变量的同时进行。

```c++
struct SPerson{
	char name[20];
	char sex;
	int age;
};
SPerson p = {"Cheng", 'F', 20};	// 正确
```

结构体是类的特例：结构体中的成员的默认访问权限是公有的，类中的成员的默认访问权限是私有的。

定义对象初始化时，**私有数据成员在定义时不能通过赋初值的方式初始化**。

```C++
struct CPerson{
	char name[20];
	char sex;
	int age;
};
CPerson p = {"Cheng", 'F', 20};	// 错误
```



那么，对象的私有数据成员能否在定义对象的同时进行初始化？ —— 能！通过C++提供的构造函数完成。



▪️ 构造函数

> 构造函数的功能：在创建对象时，由系统自动调用，用给定的值对数据成员初始化。
>
> 必须是public属性的。

可定义一个或多个构造函数（构造函数的重载），构造函数的一般格式：

`ClassName(<形参列表>){...}`								// 类内定义，末尾没有分号

`ClassName::ClassName(<形参列表>){...}`			//类外定义，末尾没有分号

🔺构造函数的特点：函数名与类名相同，且不给出返回值类型。

🔺调用时机：在创建对象时，由系统自动调用。



▪️构造函数实例

```C++
// 头文件date.h 定义日期类Date
#include<iostream>
using namespace std;
class Date{
  	int m_Year, m_Month, m_Day;
  public:
  	Date(){ m_Year = 2016, m_Month = 5, m_Day = 1; }		// 重载构造函数1
  	Date(int y){ m_Year = y, m_Month = 5, m_Day = 1; }	// ~2
  	Date(int y, int m){ m_Year = y, m_Month = m, m_Day = 1; }	// ～3
  	Date(int y, int m, int d){ m_Year = y, m_Month = m, m_Day = d; }	// ~4
  
  	void ShowDate(){
      cout<<m_Year<<"."<<m_Month<<"."<<m_Day<<endl;
    }
};
```

```c++
#include"date.h"
int main(){
	Date d1;								// 自动调用构造函数1
  Date d2(2016);					// 自动调用构造函数2
  Date d3(2016, 10);			// 自动调用构造函数3
  Date d4(2016, 10, 1);		// 自动调用构造函数4
  ....
}
```

🔺定义对象时，系统根据你的传参决定调用哪个构造函数。

以上四个构造函数可简化为一个(带参数缺省值的构造函数)。

```c++
Date(int y = 2016, int m = 5, int d = 1){
	m_Year = y; m_Month = m; m_Day = d;
}
```



▪️构造函数的特点：

1. 构造函数是成员函数。
2. 构造函数的函数名与类名相同，且不给出返回值类型。意味着：
   1. 不管是声明还是定义，函数名前面都不能出现返回值类型，即使是 void 也不允许；
   2. 函数体中不能有 return 语句。
3. 构造函数可以重载，但是创建对象时只有一个构造函数会被调用。
4. 一般将构造函数定义为公有成员函数。
5. 创建对象时，系统自动调用构造函数。
6. 一旦在类中定义了构造函数，那么创建对象时就一定要调用，不调用是错误的。
7. 不能通过对象名调用构造函数，如d1.Date(2001)是错误的。



▪️缺省构造函数/默认构造函数

如果用户自己没有定义构造函数，那么编译器会自动生成一个默认的构造函数，只是这个构造函数的函数体是空的，也没有形参，**也不执行任何操作**。比如上面的 Date 类，**默认生成的构造函数如下：**

```
Date(){}
```

一个类必须有构造函数，要么用户自己定义，要么编译器自动生成。一旦用户自己定义了构造函数，不管有几个，也不管形参如何，编译器都不再自动生成。**即系统自动生成缺省函数的前提是：编译者不定义任何构造函数。**

🔺自己定义的缺省构造函数有两种：

```c++
// 1.所有参数均带有缺省值的构造函数
Date(int y = 2000, int m = 1, int d = 1)
{m_Year = y; m_Month = m; m_Day = d;}

// 2.没有参数的构造函数
Date(){m_Year = 2000; m_Month = 1; m_Day = 1;}
```

一般地，这两种缺省构造函数只能写一个。

▪️析构函数

> 析构函数的功能是系统在撤销对象时，自动调用，做一些清理工作。

定义析构函数的格式：

`~ClassName(){...}`		// ClassName是类名，析构函数名为～ClassName

`ClassName::~ClassName(){...}`		// 类外定义



▪️析构函数的特点：

1. 一种特殊的成员函数，一般为公有成员函数。

2. 没有返回值，一般由系统自动调用，而是在销毁对象时自动执行。构造函数的名字和类名相同，而析构函数的名字是在类名前面加一个`~`符号。

3. 析构函数没有参数，不能被重载，因此一个类只能有一个析构函数。

4. 如果用户没有定义，编译器会自动生成一个默认的析构函数。

   

▪️析构函数被自动调用的两种情况：

1. 当对象是系统自动创建的，**在对象的作用域结束时**，系统自动调用析构函数。先调用析构函数再撤销该对象本身的存储空间。
2. 若对象是使用new运算符动态创建的，则在**使用delete运算符撤销它时**，自动调用析构函数。



▪️缺省析构函数

> 在定义类时，如果编译者没有定义析构函数，则编译器自动生成一个缺省析构函数。

`ClassName::~ClassName(){}`		// 该函数体为空，不做任何工作。

🔺注意：

1. 在撤销对象时，若不需要做任何结束工作，可以不显式定义析构函数
2. 当类中有动态申请（new）的存储空间时，必须显式定义析构函数。



C++中的 new 和 delete 分别用来分配和释放内存，它们与C语言中 malloc()、free() 最大的一个不同之处在于：用 new 分配内存时会调用构造函数，用 delete 释放内存时会调用析构函数。构造函数和析构函数对于类来说是不可或缺的，所以在C++中我们非常鼓励使用 new 和 delete。



▪️调用构造函数和析构函数的时机

1. 全局变量（总是静态的）：在程序（main()函数）开始执行时，创建对象，系统自动调用构造函数；程序结束时，系统在撤销对象前自动调用析构函数。
2. 局部动态变量：当程序执行进入作用域，在定义对象处系统自动创建对象，调用构造函数；退出作用域时，系统在撤销对象前调用析构函数。
3. 局部静态变量：在程序执行流程首次到达定义对象处，系统自动调用构造函数；程序（main()函数）结束时，系统在撤销对象前自动调用析构函数。
4. 动态申请的对象：使用new创建对象时，系统自动调用构造函数；使用delete撤销对象时，系统自动调用析构函数。

🔺所有的静态对象的撤销顺序（调用析构函数的顺序）与创建顺序（调用构造函数的顺序）相反。——— ==因为是栈空间存储==

▪️拷贝构造函数

> 特殊的构造函数。用一个已知对象来初始化一个新创建的同类对象。

定义格式：

`ClassName::ClassName( [const] ClassName & Obj ){<函数体>}`	// 注意形参是实参对象的[常]引用

const：如果给出，表示形参Obj在拷贝构造函数的函数体中是对象常量，不能被修改。



▪️通过实例说明拷贝构造函数的作用。

```c++
// 定义point.h的头文件
class Point(){
  	int x, y;
  public:
  	Point(int a = 0, int b = 0)	//定义缺省构造函数
    { x = a; y = b; }
  	Point(Point &p);		//拷贝构造函数原型声明
  	～Point()
    { cout<<x<<','<<" Destructor Called. \n"; }
  	void Show()
    { cout<<"Point: "<< x << ',' << y <<endl;}
  	int Getx()
    { return x;}
  	int Gety()
    { return y; }
};
Point::Point(Point &p)
{ x = p.x; y = p.y;
 	cout<<x<<','<<y<<" Copy-initialization Constructor Called. \n";
}
```

🔺此处p.x可以访问对象p的私有成员x，原因：还是在类内，可以直接访问私有对象成员。

```c++
#include<iostream>
using namespace std;
#include "point.h"
int main(){
  Point p1(6,8), p2(4,7);
  Point p3(p1);		// A
  Point p4 = p2;	// B
  p1.Show();
  p2.Show();
  p3.Show();
  p4.Show();
  return 0;
}
```
