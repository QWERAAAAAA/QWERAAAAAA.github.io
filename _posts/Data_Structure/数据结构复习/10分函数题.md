### 顺序表

```C++
// 存储结构
typedef struct{
	ElemType *elem;
	int length;
}SqList;

// 初始化：建立一个空表
Status InitList(SqList &L){
  L.elem = new ElemType[MAXSIZE];		// L是结构体类型，[用点]L.elem
  if(!L.elem) exit(OVERFLOW);
  L.length = 0;
  return 0;
}

// 
```
