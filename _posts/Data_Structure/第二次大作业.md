## 问题描述

设计一个简单计算器。给定一个包含多位数加减乘除运算的的算术表达式，请你编程计算表达式的值。

注：计算器可进行多次计算，直到主动结束程序才退出。

## 输入格式

输入你需要计算的表达式，表达式中只包含数字、运算符和括号，所有参与运算的数字均为整数。

## 输出格式

一个整数，表示这个表达式的值。

## 输入输出样例

样例1:

```in
(8+12)*5
```

```out
100

```

样例2:

```in
((8*8-20)+78)*3+2
```

```out
368

```

## 说明/提示

对于 30%30%的数据，0≤0≤表达式中加法运算符和乘法运算符的总数≤100≤100；

对于 80%80%的数据，0≤0≤表达式中加法运算符和乘法运算符的总数≤1000≤1000；

对于100%100%的数据，0≤0≤表达式中加法运算符和乘法运算符的总数≤100000≤100000。
