#### 一、struct的定义

定义方式：

1. 定义结构体类型，再定义结构体变量；
2. 定义结构体类型的同时定义结构体变量；
3. 直接定义结构体变量。

```
// 1.

struct date{
	int month;
	int year;
};
struct date a;

// 2.

struct date{
	int month;
	int year;
}a;	// a为结构体变量

// 3.

struct {
	int month;
	int year;
}a;			//不能再定义新的结构体变量了，通常用于只需要一个结构体变量的情况。

```

#### 二、typedef

定义方式：

1. typedef 类型名 新类型名
2. typedef 类型定义 新类型名（变量的名字）

```
typedef struct{
	int month;
	int year;
}SqList;	// SqList为新类型名
SqList *l;	// 这样定义了一个指向结构体的指针变量
```
