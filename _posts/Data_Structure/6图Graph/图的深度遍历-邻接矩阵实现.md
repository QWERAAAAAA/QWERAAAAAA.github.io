### 函数接口定义：

```c++
void DFS(MGraph G,Vertex v);
```

其中MGraph是邻接矩阵存储的图，定义如下：

```
#define MaxVertexNum 10     /*定义最大顶点数*/
typedef int Vertex;/* 用顶点下标表示顶点,为整型 */
typedef struct{
    int arcs[MaxVertexNum][MaxVertexNum]; /*邻接矩阵*/
    int vexnum,arcnum; /*图中的顶点数vexnum和边数arcnum*/
}MGraph; /*用邻接矩阵表示的图的类型*/
```

### 问题：

1. 怎么确定第一个邻接点和下一个邻接点？

   

2. 怎么实现既能满足连通图，同时满足非连通图？



---

### 我的想法：

1. 邻接点是二维数组i和j的关系，所以只要找到那个为1的【i】【j】，把它的j赋值给w作为下一个顶点（下一个递归的开始）
2. 把连通图套进非连通图的循环中。



接下来是怎么实现这些步骤？

大佬答案

```
void DFS(MGraph G,Vertex v){//图G为邻接矩阵类型 
    int w;
    visited[v] = 1;//访问第v个顶点，并置访问标志数组相应分量值为true（1）
    printf(" %d",v);
    for(w = 0; w < G.vexnum; w++)//依次检查邻接矩阵v所在的行
        if((G.arcs[v][w] != 0)&& (!visited[w]))  DFS(G, w);
        //G.arcs[v][w]!=0表示w是v的邻接点，如果w未访问，则递归调用DFS 
}
```

理解：

1. 直接用列（顶点的邻接点集合）作为循环，既可以判断【i】【j】是否为1（是否有边），又可以判断这个新的顶点是否已经访问过了（visited数组）
2. 为什么这个“ %d”要有个空格啊 —— 输出格式要求哈哈。
