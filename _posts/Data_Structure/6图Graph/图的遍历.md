> 图的遍历：从图中某一顶点出发，按某种方法对图中所有顶点进行访问且仅访问一次
>
> ⬇️
>
> 出现问题：一个连通图就是一个连通分量——>可以生成一个树。
>
> ​					非连通图生成不了一个树，有的点访问不到。
>
> ⬇️
>
> 解决方案：
>
> 1. 为了避免同一顶点被访问多次：
>    - 记下每一个已经访问过的顶点（设置一个辅助数组visited[n]）
>    - 初始值置为false/0
>    - 一旦访问了顶点vi
>    - 把visited[i] 置true/1
> 2. 针对非连通图：
>    - 需要对所有顶点循环，判断它是否被访问过，对未访问过的使用连通图的遍历算法
>    - 每一次调用则会遍历完一个连通分量，有多少次调用，就说明图中有多少个连通分量。

## 算法实现

> 两个关键：1⃣️存储结构（邻接矩阵or邻接表）2⃣️标志数组visited[n]



### DFS遍历连通图

**简洁版**

```c
bool visited[MVNum];		
void DFS(Graph G,int v){		
	cout<<v;		
	visited[v] = true;		
	for(w = FirstAdjVex(G,v);w >= 0; w = NextAdjVex(G,v,w))
		if(!visited[w])		
			DFS(G,w);
}
```

**解析版**

```c
bool visited[MVNum];		// 设置标志数组（静态变量，系统自动赋值全为0）
void DFS(Graph G,int v){		// 从第v个顶点出发递归地深度优先遍历图
	cout<<v;			// 访问第v个顶点
	visited[v] = true;			// 置标志数组为1
	for(w = FirstAdjVex(G,v);w >= 0; w = NextAdjVex(G,v,w))
	// 目的：依次检查v的所有邻接点w
	// FirstAdjVex(G,v)表示v的第一个邻接点
	// NextAdjVex(G,v,w)表示v相对于w的下一个邻接点
	// w>=0 表示存在邻接点
		if(!visited[w])			// 对v的尚未访问的邻接点w递归调用DFS()
			DFS(G,w);
}
```



### DFS遍历非连通图

```c
void DFSTraverse(Graph G){
	for(v = 0; v < G.vexnum; ++v)
		visited[v] = false;
	for(v = 0; v < G.vexnum; ++v)
		if(!visited[v])
			DFS(G,v);
}
```

### ⭕️ 邻接矩阵的DFS

**简洁版**

```c
void DFS_AM(AMGraph G,int v){
	int w;
	cout<<v;
	visited[v] = 1;
	for(w = 0; w < G.vexnum; w++)
		if((G.arcs[v][w] != 0) && (!visited[w]))
			DFS_AM(G,w);
}
```

**解析版**

```c
void DFS_AM(AMGraph G,int v){
	int w;
	cout<<v;
	visited[v] = 1;
	for(w = 0; w < G.vexnum; w++)	// 只需依次检查该顶点所在的行就行了
		if((G.arcs[v][w] != 0) && (!visited[w]))
		// （G.arcs[v][w]!= 0）表示w是v的邻接点
		// 如果w未被访问，则递归调用DFS_AM（）
			DFS_AM(G,w);
}
```
