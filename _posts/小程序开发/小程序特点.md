小程序特点概述

1. 没有DOM
2. 组件化开发：具有特定功能效果的代码集合
3. 体积小，单个压缩包体积不能大于2M，否则无法上线
4. 小程序的四个重要文件
   1. *.js
   2. *.wxml ——> view结构 ——> html
   3. *.wxss ——> view样式 ——> css
   4. *.json ——> view数据 ——>json文件（配置文件）
5. 小程序的适配方案：rpx/响应式像素单位
   1. 小程序适配单位：rpx
   2. 小程序规定任何屏幕下宽度为750rpx
   3. 小程序会根据屏幕的宽度不同自动计算rpx值的大小
   4. iPhone6下 1rpx = 1物理像素 = 0.5px （各机型下需要换算）
