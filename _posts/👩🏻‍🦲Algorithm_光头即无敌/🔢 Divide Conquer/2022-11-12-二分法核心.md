# 二分法

> 基本原理：从中间开始查找，小了往左边找，大了往右边找。
>
> 明白了基本思路，然后牢牢记住代码实现模版即可。

## 一、搜索和目标值相等的数

这一类是最简单的，例如对于数组 arr = {1, 2, 5, 6, 9}，要我们搜索返回目标数 target = 6，这个时候我们需要返回 6 的下标 i = 3。代码如下

```c++
int BS1(int arr[], int target, int len){        // 查找和目标值相等的数
    int left = 0;
    int right = len - 1;
    while (left <= right){
        int mid = left + (right - left) / 2;
        if(arr[mid] == target)
            return mid;
        else if(arr[mid] < target){
            left = mid + 1;
        }
        else
            right = mid - 1;
    }
    return -1;
}
```

不过这个有一个需要注意的点，就是很多人在求 mid 的时候，会这样写

`mid = (left + right) / 2;`

其实这样写是有点小问题的，**因为 left + right 有可能导致数值溢出**，从而 mid 的计算就错误了，所以经常这样写的小伙伴要注意了，严谨的写法应该是这样写：

`mid = left + (right - left) / 2;`



## 二、 查找第一个不小于目标值的数

查找第一个**不小于（大于等于）**目标的数，我觉得这类出现的频率是最高的，而且要注意，目标数并不一定就出现在数组中。

例如对于数组 arr = {1, 2, 5, 6, 9}，目标数 target = 6，那么我们要返回下标 i = 4 （这里应该返回i = 3）。

又如 arr = {0, 1, 2, 2, 2, 3}，target = 2。我们要返回 i = 2。

代码如下

```c++
int BS2(int arr[], int target, int len){        // 查找第一个不小于目标值的数
    int left = 0, right = len - 1;
    while(left <= right){
        int mid = left + (right - left) / 2;
        if(arr[mid] < target)
            left = mid + 1;
        else{
            right = mid;
            return right;
        }  
    }
    return -1;
}
```

在代码上，和第一种最大的区别就是 `right = mid - 1 变成 right = mid` 了。

对了，还有一种和**查找第一个不小于目标值的数**类似的题，就是**找最后一个小于目标值的数**。这很简单，直接返回 right - 1 就可以了。



## 三、查找第一个大于目标值的数

查找第一个大于目标的数，我觉得这类出现的频率也是最高的，而且也要注意，目标数并不一定就出现在数组中。

例如对于数组 arr = {1, 2, 5, 6, 9}，目标数 target = 6，那么我们要返回下标 i = 5 。(这里应该是i = 4)

又如 arr = {0, 1, 2, 2, 2, 3}，target = 2。我们要返回 i = 5。

我们先来看下代码

```c++
int BS3(int arr[], int target, int len){        // 查找第一个大于目标值的数
    int left = 0, right = len - 1;
    while (left <= right){
        int mid = left + (right - left) / 2;
        if(arr[mid] <= target)
            left = mid + 1;
        else{
            right = mid;
            return right;
        }
    }
    return -1; 
}
```

有木觉得和第二种类型的代码太像了，`只需要在 if 语句里吧 arr[mid] < target 改成 arr[mid] <= target 即可。`至于为啥？因为**第一个不小于**和**第一个大于等于**是同一个意思，但是这道题是**第一个大于**，不包括**等于**这种情况啊。所以把 < 改为 <= 即可。

## 总结

其实对于二分法的查找，有很多种类型，不过，代码的差别都不大，就是有可能大于改成大于等于。`left/right = mid + 1 变成 left/right = mid`。所以有时也很容易出错，特别是脑子乱了的时候，所以我这里建议，**选择一种自己喜欢的代码风格**，然后每次做题的时候，的用你心中的那份模板代码。
