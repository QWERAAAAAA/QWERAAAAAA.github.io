## 1. Java语言概述

- **Java是一种面向Internet的编程语言。**Java applet可以在Web浏览器中运行。applet内嵌在HTML代码中。
- **Java VS C++ ：**
  - Java舍弃了C中容易引起错误的指针（以引用取代）、运算符重载、多重继承（以借口取代）
  - ==**增加了垃圾回收器功能。**==（仍会出现内存泄漏和内存溢出的问题）

- **Java是体系结构中立的。**Java程序（后缀为java）可以在Java平台被编译为体系结构中立的**字节码格式**（后缀为class），然后可以在实现这个Java的平台的任何系统中运行。
- **Java语言是解释型的。**

### 03 Hello World！

> 👏迈向大佬之路的第一步！

```java
public class HelloWorld{
  public static void main(String[] args){
    System.out.println("Hello World!")
  }
}
```

**分析：**

- 每个Java程序都是一套类的定义集合。Java程序中的全部内容都必须放在类中。（强制面向对象）
- 方法包含在类中，每个可执行的Java程序必须有一个main方法，main方法是程序开始执行的入口。
- <img src="https://tva1.sinaimg.cn/large/008vxvgGgy1h85mxgnwx6j31zu0rsn0a.jpg" alt="image-20221115110818535" style="zoom: 20%;" />

2. 通过**javac命令**对该Java文件进行**编译**。

   ​	`javac HelloWorld.java`

3. 通过**java命令**对生成的class文件进行**运行**。

​				`java HelloWorld`

## 3. 标识符

> 凡是可以自己起名字的地方都叫做标识符。

**规范：**

- **包名**：字母全部小写。xxxyyyzzz
- **类名、接口名**：所有首字母大写。XxxYyyZzz
- **变量名、方法名**：多单词时，第一个首字母大写，之后的首字母大写。xxxYyyZzz
- **常量名**：所有字母大写，多单词用下划线连接。XXX_YYY_ZZZ

### 02 变量声明与初始化

- **变量声明**告知编译器根据数据类型为变量分配合适的内存空间。

  ```java
  int count;
  double radius;
  ```

- 如果**几个变量为同一类型**，允许一起声明

  ```java
  int i, j, k;
  ```

- 可以一步完成变量的声明和初始化

  ```java
  int count = 1;
  ```

- 它等同于下面的两条语句

  ```java
  int count;
  count = 1;
  ```

- 可以使用简捷的方式来同时声明和初始化**同一类型的变量**

  ```java
  int i = 1, j = 2;
  ```

🗣提示：**方法中声明的变量在使用前必须被赋值**。在任何时候，都要尽可能一步完成变量的声明和赋初值。



### 03 基本数据类型变量的转换

**自动类型转换**：容量小的类型自动转换为容量大的数据类型。

![截屏2022-11-13 15.24.38](https://tva1.sinaimg.cn/large/008vxvgGgy1h83j3i3esdj315406uaau.jpg)



**注意：**

1. boolean类型不能与其它数据类型运算
2. 当把任何基本数据类型的值**和字符串(String)进行连接运算时(+)**，基本数据类型的值将自动转化为字符串(String)类型。



**强制类型转换**：通常是将容量大的数据类型转换为容量小的数据类型

使用时要加上强制转换符：( )，但**可能造成精度降低或溢出,**格外要注意。 

- 通常，字符串不能直接转换为基本类型，但通过**基本类型对应的包装类**则可以实现把字符串转换成基本类型。 
- 如： String a = “43”; int i = Integer.parseInt(a);





## 5. 常量

> 常量有三种类型：静态常量、成员常量和局部常量。

定义常量—— final关键字

```java
public class HelloWorld {
    // 静态常量PI
    public static final double PI = 3.14;
    
  	// 声明成员常量y
    final int y = 10;

    public static void main(String[] args) {
        
      	// 声明局部常量x
        final double x = 3.3;
    }
}
```

- **静态常量PI：**使用在 final 之前 public static 修饰。public static 修饰的常量作用域是全局的，不需要创建对象就可以访问它，在类外部访问形式为 `HelloWorld.PI`
- **成员常量y：**作用域类似于成员变量，但不能修改。（注意：与静态成员`static int y = 10;`不一样。
- **局部常量x：**作用域类似于局部变量，但不能修改。



在定义常量时，需要注意如下内容：

- 在定义常量时就需要对该常量进行初始化。
- final 关键字不仅可以用来修饰基本数据类型的常量，还可以用来修饰对象的引用或者方法。
- 为了与变量区别，常量取名一般都用大写字符。


当常量被设定后，一般情况下不允许再进行更改，如果更改其值将提示错误。

### 01 赋值=

> 这个玩法好多啊～

==**玩法1: 赋值表达式**==

```java
System.out.print(x = 1);		// 是正确的
```

- 它等价于:

```java
x = 1;
System.out.print(x);
```



==**玩法2: 给多个变量赋值**==

```java
i = j = k = 1;
```

- 它等价于：

  ```java
  k = 1;	j = k;	i = j;
  ```



⚠️ 赋值语句中，左边变量的数据类型必须与右边值的数据类型兼容。`int x = 1.0` 是非法的。

✍🏼**if-else if-else的技巧：**

- 当多个条件是“互斥”关系时，条件判断语句及执行语句间顺序无所谓

- 当多个条件是“包含”关系时，**“小上大下 / 子上父下”**。（这个if中的条件不满足才会往下一个去）

```java
public class AgeTest{
  public static void main(String args[]){
    int age = 38;
    if(age < 0)
      System.out.println("不可能！");
    else if(age < 18)
      System.out.println("还未成年！不能雇用");
    else if(age < 60)
      System.out.println("还未到退休年龄,想退休还早着呢");
    else
      System.out.println("人家芳龄" + age + "正在退休养老～");
  }
}
```
