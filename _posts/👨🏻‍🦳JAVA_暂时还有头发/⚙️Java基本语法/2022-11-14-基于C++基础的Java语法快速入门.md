## Java程序设计基础

### 01 标识符

> 凡是可以自己起名字的地方都叫做标识符。

**规范：**

- **包名**：字母全部小写。xxxyyyzzz
- **类名、接口名**：所有首字母大写。XxxYyyZzz
- **变量名、方法名**：多单词时，第一个首字母大写，之后的首字母大写。xxxYyyZzz
- **常量名**：所有字母大写，多单词用下划线连接。XXX_YYY_ZZZ



### 02 关键字与保留字

Java 语言目前定义了 51 个关键字，这些关键字不能作为变量名、类名和方法名来使用。以下对这些关键字进行了分类。

1. 数据类型：boolean、int、long、short、byte、float、double、char、class、interface。
2. 流程控制：if、else、do、while、for、switch、case、default、break、continue、return、try、catch、finally。
3. 修饰符：public、protected、private、final、void、static、strict、abstract、transient、synchronized、volatile、native。
4. 动作：package、import、throw、throws、extends、implements、this、supper、instanceof、new。
5. 保留字：true、false、null、goto、const。

### 04 常量

> 常量有三种类型：静态常量、成员常量和局部常量。

定义常量—— final关键字

```java
public class HelloWorld {
    // 静态常量PI
    public static final double PI = 3.14;
    
  	// 声明成员常量y
    final int y = 10;

    public static void main(String[] args) {
        
      	// 声明局部常量x
        final double x = 3.3;
    }
}
```

- **静态常量PI：**使用在 final 之前 public static 修饰。public static 修饰的常量作用域是全局的，不需要创建对象就可以访问它，在类外部访问形式为 `HelloWorld.PI`
- **成员常量y：**作用域类似于成员变量，但不能修改。（注意：与静态成员`static int y = 10;`不一样。
- **局部常量x：**作用域类似于局部变量，但不能修改。



在定义常量时，需要注意如下内容：

- 在定义常量时就需要对该常量进行初始化。
- final 关键字不仅可以用来修饰基本数据类型的常量，还可以用来修饰对象的引用或者方法。
- 为了与变量区别，常量取名一般都用大写字符。


当常量被设定后，一般情况下不允许再进行更改，如果更改其值将提示错误。



### 05 变量

> Java是强类型的语言：所有的变量必须先声明、后使用；指定类型的变量只能接受类型与之匹配的值。



- 声明变量

  ```java
  String employee;
  ```

- 变量赋值

  ```java
  char usersex = 'female';		// 声明变量的同时，赋值
  
  String username;		// 先声明，后赋值
  username = "Alan";
  ```

- 多个**同类型**的变量可以同时定义或者初始化

  ```java
  String username,address,phone,tel;    // 声明多个变量
  int num1=12,num2=23,result=35;    // 声明并初始化多个变量
  ```

- 作用域：静态变量、全局变量、局部变量
